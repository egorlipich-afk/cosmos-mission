📡 Сюжет: петля времени на станции «Алгоритм»

Ты — капитан исследовательского корабля «Алгоритм». Энергетический щит повреждён, и станция застряла во временной петле. Чтобы вырваться, необходимо восстановить алгоритмы циклов с условием: цикл с предусловием (while), цикл с постусловием и тело цикла. Используй логику из учебника информатики 8 класса (параграф 2.4.3, страницы 81-91).

  • 🔄 Цикл с предусловием (цикл-ПОКА, while) — сначала проверяется условие, затем выполняется тело цикла (страницы 10-13)
  • 🔁 Цикл с постусловием — сначала выполняется тело цикла, потом проверяется условие окончания (страница 15, пример с четверостишием)
  • ⚙️ Тело цикла — последовательность действий, многократно повторяющаяся в процессе выполнения цикла (страницы 2-5)
  • ⛔ Зацикливание и выход из цикла — условие продолжения должно изменяться внутри тела, иначе цикл станет бесконечным
  • 📘 Мнемоническая история: спортсмен увеличивает дистанцию пробега на 10 процентов каждый день, пока не достигнет 25 километров (страница 17), и робот-художник, рисующий фигуры заданное количество раз
